# CocoRunning
Sports card game collection game

#实践项目

一、项目基础

  基于高德地图API、bmob后端云数据库、bmob即时通信API、U3D。
  
  项目采用了微服务架构，数据存储于云端。本地服务进行模糊压缩后定期存储更新数据，进行重要数据的容灾备份。
  
  项目代码架构采用mvp模式
  
  服务器端有相应的路径分配算法和图像目标检索算法的实现，此外还有基于图像语意分割的城市景观和安全系数实现
